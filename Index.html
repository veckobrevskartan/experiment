OIAT-kriterier

OIAT-kriterierna används för att kvalitetssäkra öppna källor och analys. [Här beskrivs varje kriterium – t.ex. relevans, aktualitet, trovärdighet – på svenska.] Varje kriterium kan expandera till mer information.

Indikatorer

Indikatorer är tidiga varningssignaler som pekar på kommande skeenden:contentReference[oaicite:8]{index=8}. Exempelvis kan ovanliga inköp eller ändrat kommunikationsmönster vara indikatorer. I denna presentation kan man klicka för att visa fler exempel på indikatorer.

Kända och okända

Analysen tar också höjd för Rumsfelds modell med ”kända kända”, ”kända okända” och ”okända okända”:contentReference[oaicite:9]{index=9}. Det innebär att vi definierar vad vi säkert vet, vad vi vet att vi inte vet, och vad vi ännu inte kan förutse.

Produktformat

Underrättelseprodukter kan utformas som rapporter, lägesbilder, dashboards eller bulletiner. Här finns exempelgrafik för varje formatstyp – klicka på ett format för helskärmsgrafik.

Interaktiva diagram

Diagrammen visar händelser per kategori, tidslinje per månad och kategori vs månad (heatmap) från data i `data.js`.

Händelsekarta

Klicka på markörer för detaljer. Kartan är filterbar per kategori och kan visas helskärm.
